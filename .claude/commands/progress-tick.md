# /progress-tick

推进 PROGRESS.md 中当前任务的下一步，将当前步骤标记为完成，下一步骤标记为"← 当前"。

## 执行步骤

1. 读取 PROGRESS.md，找到"状态：进行中"的任务
2. 找到第一个 `- [ ]` 条目，将其改为 `- [x]`
3. 更新"当前进度：第 N 步 / 共 M 步"
4. 将下一个 `- [ ]` 条目后追加 `（← 当前）`
5. 如果所有步骤都完成，将任务状态改为"已完成"，并在 docs/progress.md 追加变更日志
6. 更新"下一步行动"字段

## 注意

- 推进后评估是否需要更新 BLUEPRINT.md（若有代码结构变化）

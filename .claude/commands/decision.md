# /decision <标题>

向 DECISIONS.md 追加一条新的架构决策记录（ADR）。

## 执行步骤

1. 使用以下模板，在 DECISIONS.md 末尾追加新条目
2. 将 `<标题>` 替换为用户提供的参数
3. 填写当前日期（YYYY-MM-DD）
4. 询问用户填写各字段内容，或根据上下文推断后请用户确认

## 模板

```
## [YYYY-MM-DD] <标题>
- **背景**：当时面临什么问题
- **备选方案**：
  - 方案 A：...（利弊：...）
  - 方案 B：...（利弊：...）
- **决定**：选了哪个
- **原因**：为什么
- **后果**：这个选择带来的限制和后续影响
- **状态**：active
```

## 注意

- 记录完成后更新 DECISIONS.md 末尾的 `> 最后更新` 日期

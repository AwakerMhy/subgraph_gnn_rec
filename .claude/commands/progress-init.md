# /progress-init <任务名>

在 PROGRESS.md 中初始化新任务区块。

## 执行步骤

1. 在 PROGRESS.md 顶部（最近任务区域）追加新任务区块
2. 使用当前时间填写"开始时间"
3. 根据用户描述或 TODO.md 中的任务条目拆分子步骤

## 模板

```
## [<任务名>]
- 状态：进行中
- 开始时间：YYYY-MM-DD HH:MM
- 当前进度：第 1 步 / 共 N 步
- [ ] 步骤1：描述（← 当前）
- [ ] 步骤2：描述
- [ ] 步骤3：描述
- 已变更文件：（待填写）
- 下一步行动：步骤1 具体描述
```

## 注意

- 如果当前有未完成任务，先将其状态标记为"已挂起"再新建

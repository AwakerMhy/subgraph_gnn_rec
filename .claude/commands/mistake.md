# /mistake <标题>

向 MISTAKES.md 追加一条错误记录。

## 执行步骤

1. 在 MISTAKES.md 末尾追加以下模板
2. 将 `<标题>` 替换为用户提供的参数
3. 填写当前日期
4. 根据上下文填写各字段，或询问用户补充

## 模板

```
## [YYYY-MM-DD] <标题>
- **类别**：思路错误 / 代码错误 / 工具误用 / 需求误解
- **现象**：发生了什么、报错信息或用户原话
- **根因**：为什么会犯（追到底层假设）
- **教训**：下次类似情况应该怎么做
- **检测信号**：什么样的代码/任务/上下文应该触发警觉
- **关联文件**：（可选）
- **复发次数**：1
```

## 注意

- 如果 MISTAKES.md 中已有类似条目，检查是否应该增加"复发次数"而非新增
- 每 ~20 条触发 `/meta-reflect`
